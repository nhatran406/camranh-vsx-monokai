# CamRanh Monokai

CamRanh Monokai is a VS Code theme adapted from the `camranh-vsx-monokai` visual language and keeps the same full color palette, contrast, and font pairing.

## Included

- 1 dark theme: `CamRanh Monokai`
- Original palette preserved exactly

## Install Locally

1. Open this folder in VS Code.
2. Run the extension in a Development Host window.
3. Select `CamRanh Monokai` from the Color Theme picker.

## Notes

This package renames the theme and project identity to CamRanh Monokai while intentionally preserving the original theme colors and typography recommendations.
